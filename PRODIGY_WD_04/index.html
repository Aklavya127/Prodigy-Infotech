Sorting Visualizer

The Sorting Visualizer is a web app that utilizes HTML, CSS, and JavaScript to visually demonstrate sorting algorithms such as Bubble Sort and Merge Sort. Bars represent elements, dynamically rearranging as the algorithm executes. The user-friendly interface allows customization of elements, animation speed, and provides options to pause or reset. Overall, it is an interactive educational tool for understanding sorting algorithms through real-time visualizations.

Speech Recognition App - Jann

Jann is an advanced Python speech recognition app with a user-friendly interface, leveraging powerful algorithms such as acoustic modeling, language modeling, and hidden Markov models. Trained on extensive datasets, it ensures precise transcription of various languages, including HTML, CSS, Python, Java, JavaScript, and Artificial Intelligence. Users can capture audio input, undergo preprocessing, and benefit from features like text formatting and punctuation correction, enhancing the overall speech-to-text experience.
